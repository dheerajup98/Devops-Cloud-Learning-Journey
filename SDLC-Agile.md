# ğŸš€ What is SDLC?

**SDLC (Software Development Life Cycle)** is a structured process used to design, develop, test, deploy, and maintain software systems.

It defines:

- How software is built  
- Who works on each stage  
- How quality is maintained  
- How risk is minimized  
- How software moves to production  

SDLC ensures **systematic, reliable, and scalable software delivery**.
